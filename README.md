nj is a build tool for c/c++ projects. nj is based on ninja build system. Instead of generating ninja build files, it
execute ninja build dependencies on the fly.

Build from source
-----------------

1. Clone the repository
2. Run `bun make.js` in the project directory.

When nj is run, it executes build.js in the same directory as nj.

There's a sample build.js in the project directory.
